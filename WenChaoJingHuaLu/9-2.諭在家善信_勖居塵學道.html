# 乙、勖居塵學道

# 勖居塵學道1

居塵學道，即俗修真，乃達人名士，及愚夫愚婦，皆所能為。勉力修持，以在家種種系累，當作當頭棒喝。長時生此厭離之心，庶長時長其欣樂之志。即病為樂，即塞成通。上不失高堂之歡，下不失私室之依。而且令一切人同因見聞，增長淨信，何樂如之。（正）復周群錚書五

居於紅塵，學修佛道，即於俗世，而修真諦。這是達人名士，以及普通百姓，都能夠去做的。努力修持，以在家的種種系縛煩累，當作當頭棒喝。長時間生起這種厭離之心，使得長時間增長這種欣樂求生極樂的志向。即病患而成為良藥，即堵塞而成為通暢。上不失高堂慈母的歡心，下不失私室妻子的依靠。而且令一切人，同因所見所聞，而增長淨信。這是何等快樂啊！

# 勖居塵學道2

人生世間，不可無所作為。但自盡誼盡分，決不於誼分之外，有所覬覦。士農工商，各務其業，以為養身養家之本。隨分隨力，執持佛號，決志求生西方。凡有力能及之種種善事，或出資，或出言，為之贊助。否則發隨喜心，亦屬功德。以此培植福田，作往生之助行。如順水揚帆，更加艣棹，其到岸也，不更快乎。（正）復寧波某居士書

人生世間，不可以無所作為。但是只要盡好自己的道誼本分，決不在自己的道誼本分之外，另外有所企圖。士農工商，各自做好自己的行業，作為養活自身家庭的本錢。隨自己的本分能力來執持佛號，決心求生西方。凡是有能力做到的種種善事，或者出錢，或者說話發言，來加以贊助。再不然，發隨喜心，也是屬於功德。以此來培植福田，作為往生的助行。如順水揚帆，更加上櫓棹的幫助，這個船要到岸，不是更加快速了嗎？

# 勖居塵學道3

所言培植功德，當以開人知識者為第一。現今增廣文鈔已經排完，尚未結收。令弟若肯任若干，自己施送。俾一切閱者知往生淨土之所以然，以此功德，為其父作往生之券，加以至誠，必可如願。是為最有利益真實功德。（三）復徐蔚如書

所說的培植功德，應當以開啟他人的智慧見識的功德為第一。現今《增廣文鈔》已經排完，還沒有完結收尾。你弟弟如果肯任資若干本，自己布施贈送。使得一切讀者，知道往生淨土的所以然。以此功德，作為他父親作往生西方的契據憑證。加上至誠心，必定可以如願。這是最有利益的真實功德。

# 勖居塵學道4

若大通家，則禪淨雙修，而必以淨土為主。若普通人，則亦不必令其遍研深經奧論，但令諸惡莫作，眾善奉行，一心念佛，求生西方即已。此人不廢居家業，而兼修出世法。雖似平常無奇，而其利益不可思議。良以愚夫愚婦，顓蒙念佛，即能潛通佛智，暗合道妙。校比大通家之卜度思量，終日在分別中弄識神者，為益多多也。（正）復謝誠明書

如果是大通家，就禪淨雙修，而必定要以淨土為主。如果是普通人，就不必令他遍研深奧的經論，只要讓他諸惡莫作，眾善奉行，一心念佛，求生西方罷了。這個人不捨棄在家的事業，而兼修出世法。雖然好似平常沒什麼奇特，而其中的利益不可思議。實在因為普通百姓，老實念佛，就能夠潛通佛智，暗合道妙。比起大通家的卜度思量，整天在分別心中玩弄識神，有利益的多啊！

# 勖居塵學道5

以閣下之才論，當依光所說，其為利益大矣。否則擇一寂靜隱晦之處，力修淨業，將從前所得之學問文章，拋向東洋大海外，作自己原是一無知識之人，於不生分別心中，晝夜六時，專持一句洪名聖號，果能死盡偷心，當必親見本來面目。從茲高豎法幢，俾一切人同歸淨土法海，生為聖賢之徒，沒預蓮池之會。方可不負所學，為大丈夫，真佛子矣。（三）復葉玉甫書

以閣下的才學而論，應當依我所說，這個利益就很大了。否則，選擇一個寂靜隱密的地方，努力專修淨業。將從前所得的學問文章，拋向東洋大海之外，當自己原本是一個無知無識的人。在不生分別心中，晝夜六時，專持一句洪名聖號。果真能夠死盡偷心，當必定親見本來面目。從此高豎法幢，使得一切人同歸淨土法海。在生成為聖賢之徒，死後預入蓮池之會。才可以不辜負所學，為大丈夫、真佛子啊！

# 勖居塵學道6

如來說法，恒順眾生。遇父言慈，遇子言孝。外盡人倫，內消情乚。使復本有真心，是名為佛弟子，豈在兩根頭發上論也。仗此好心，竭力學道，孝弟修而閭裡感化，齋戒立而殺盜潛消。研究淨土經論，則知出苦之要道。受持安士全書，則知淑世之良謨。以淨土法門諭親，以淨土法門教子，及諸親識。正以生死事大，深宜痛恤我後。不必另擇一所，即家庭便是道場。以父母兄弟妻子朋友親戚，盡作法眷。自行化他，口勸身率。使其同歸淨域，盡出苦輪。可謂戴髮高僧，居家佛子矣。（正）復林介生書一

如來說法，恆順眾生。遇父親說慈愛，遇兒子說孝順。對外要恪盡人倫道德，向內要消除情見思慮，恢復本有的真心，這名為佛弟子，哪裡是在兩根頭髮上來討論呢？何況你的家鄉偏僻在深山，知道佛法的人很少。高明的人因為語言不通的緣故，都不到那個地方。仗著這個好心，努力學道。孝順父母，友愛兄弟，感化鄉里。吃齋守戒，殺生偷盜，不知不覺消除。研究淨土經論，就知道出苦的要道。受持《安士全書》，就知道濟世的良謀。以淨土法門告知雙親，以淨土法門教導子女，以及諸位親友知識。正因為生死事大，更應該好好顧念我的將來。不必要另外找一個處所，家庭就是道場。以父母、兄弟、妻子、朋友、親戚，都成為法親眷屬。自行化他，口中勸導，以身作則，使他們同歸淨土，全部出離六道輪迴。可以說是戴髮的高僧，居家的佛子。

# 勖居塵學道7

且勿謂吾家素寒，不能廣積陰德，大行方便。須知身口意三業皆惡，即莫大之惡。倘三業皆善，即莫大之善。至如愚人不信因果，不信罪福報應，侃侃鑿鑿。依安士全書等所說，為其演說。令其始則漸信因果，繼則深信佛法，終則往生西方，了生脫死。一人如是，功德尚無量無邊，何況多人。然須躬行無玷，方可感化同人。自己妻女，能信受奉行，別人自能相觀而善矣。豈在資財多乎哉。（正）復高邵麟書一

【鑿鑿】鮮明。確實。

【躬行】親身實行。《論語·述而》：“躬行君子，則吾未之有得。”

【無玷】沒有瑕疵；不受玷污。

【同人】志趣相同或共事的人。

【相觀】觀察。

不要說我家裡一向貧寒，不能夠廣積陰德，大行方便。必須知道身口意三業都惡，就是最大的惡。倘若身口意三業都善，就是最大的善。倘若愚昧的人不相信因果，不相信罪福報應。你就從容明確、理直氣壯的，依照《安士全書》裡面所說的，為他演說。令他開始漸漸相信因果，繼而深信佛法，最終往生西方，了生脫死。能教化一個人如此，功德尚且無量無邊，何況教化很多人。然而必須自己的行為沒有瑕疵，才可以感化身邊的人。自己的妻女，能夠信受奉行，別人自然能夠互相觀察學習而向善了。哪裡在於錢財的多少呢？

# 勖居塵學道8

佛為大醫王，普治眾生身心等病。世間醫士，只能醫身。縱令著手成春，究於其人神識結果，了無所益也。汝既皈依三寶，發菩提心，為人治病。則當於醫身病時，兼寓醫心病法。何以言之，凡屬危險大病，多由宿世現生殺業而得。而有病之人，必須斷絕房事，方可速愈。欲滅宿現殺業，必須戒殺吃素，又復至誠念佛，及念觀音，則必可速愈，且能培德而種善根。儻怨業病，除此治法，斷難痊愈。其人，與其家父母妻子，望愈心急，未必不肯依從。儻肯依從，則便種出世善根，從茲生正信心，後或由此了生脫死，超凡入聖。則於彼於汝均有大益。至於斷欲一事，當以為治病第一要法。無論內證外證，病未十分復原，萬不可沾染房事。（正編文鈔壽康寶鑒序文，專治此病，宜詳閱之。編者敬注）一染房事，小病成大，大病或致立死。或不即死，已種必死之因。欲其不死，亦甚難甚難。縱令不死，或成孱弱廢人，決難保其康健。不知自己不善攝養，反說醫生無真本事。無論男女，(處女寡婦不宜說。餘俱無礙。)均當侃侃鑿鑿，說其利害，俾彼病易愈，而汝名亦因茲而彰。每每醫生只知治病，不說病忌，況肯令人改過遷善，以培德積福乎。此是市井唯利是圖之負販心行，非壽世濟人之心行。況能令人因病而得生入聖賢之域，沒歸極樂之邦之無上利益乎。古人云：不為良相，必為良醫，是以稱醫士曰大國手。世間醫士之名已高極，若兼以佛法，則藉此以度眾生，行菩薩道，實為一切各業中最要之業。以人於病時，得聞不專求利，志期利人，發菩提心之醫士所說，必能令病即愈，自不能不生正信依行也。欲人取信，切不可計謝禮多寡而生分別。儻富者認真為醫，貧者只應酬了事。久之，人皆以謀利而輕之。則所說利人之話，人亦不信從矣。又須遇父言慈，遇子言孝，兄友弟恭，夫和婦順，主仁仆忠，與因果報應之通三世，(過去。現在。未來)生死輪回之經六道。有可語者，不妨以有意作無意之閑談，使聞者漸漸開通心地。知生死輪回之可畏，幸了生脫死之有法。能如是者，誠可謂即世間法，以行佛法，由醫身病而愈心病。（續）與馬星樵書

佛為大醫王，普治眾生身心等病。世間的醫生，只能醫治身病，縱然是妙手回春，究竟對於這個人的神識結果（心靈歸宿），沒什麼利益。你既然皈依三寶，發菩提心，為人治病，那麼應當在醫治身病的時候，兼用醫治心病的方法。為什麼這麼說呢？凡是危險大病，大多由於宿世現生的殺業所得。而有病的人，必須斷絕房事，才可以快速痊癒。想要滅除宿世現生的殺業，必須戒殺吃素。又再至誠念佛，以及念觀音菩薩，那麼必定可以快速痊癒，而且能夠培植德行而種下善根。倘若是怨業病，除去這個治療的方法，斷然很難痊癒。這個人，與他家中的父母、妻子、兒女，希望他痊癒的心很急切，未必不肯依從。倘若肯依從，就種下出世的善根。從此生起正信，後來或許由此了生脫死，超凡入聖，那麼對於他，對於你，都有很大的利益。至於斷欲這件事，應當作為治病的第一重要方法。無論內症外症，病情還沒有完全恢復，千萬不可以沾染房事。一旦沾染房事，小病轉成大病，大病或許導致立即死亡。或者不會立即死亡，已經種下了必死的因種，想要他不死，也很難很難。縱然不死，或者成為一個孱弱的廢人，決定很難保住他的健康。不知道是自己不善於收攝養身，反而要說醫生沒有真本事。無論男女，（處女寡婦不適合對她們說，其餘的都沒有妨礙。）都應當理直氣壯，確確實實，敘說其中的利害，使他的病容易痊癒，而你的名聲也因此而彰顯。很多醫生只知道治病，不說病中的禁忌，何況肯讓人改過向善，來培德積福呢？這些都是市井小人，唯利是圖的小商販的存心行為，不是濟世救人的存心行為。何況能夠令人因為生病，而得以在生進入聖賢的領域，死後歸於極樂蓮邦的無上利益呢？古人說：不做好宰相，必做好醫生。所以稱醫生為大國手。世間醫生的名稱已經高到極點，如果兼以佛法教化，那麼憑藉醫術佛學來救度眾生，行菩薩道，實在是一切各種行業中最要的職業。因為一個人在生病的時候，得以聽聞到不專門是為了求利，有志期望利益他人，發菩提心的醫生所說的話，必定能夠令他的病立即痊癒，自然不能不生起正確的信心來依教奉行啊！想要使他人生起信心，千萬不可以計慮酬謝禮金的多少而生起分別心。倘若對於有錢的人，就認真為他醫治，貧窮的人，只是應酬了事，久而久之，他人都會因為你專門謀取利益而輕視你，那麼你所說利益他人的話，他人也不會相信聽從了啊！又必須遇到父親講慈祥，遇到兒子講孝順，兄長友愛，弟弟恭敬，丈夫溫和，妻子順從，主人仁厚，僕人忠誠。與因果報應通於三世，（過去、現在、未來。）生死輪迴要經過六道，有可以對他說的人，不妨以你的有心，作看視無意的閒談。使得聽到的人，漸漸開通心地，知道生死輪迴的可怕，慶幸還有個了生脫死的方法。能夠如此的話，實在可以說是，即世間法而行持佛法，由醫治身病而治愈心病。

# 勖居塵學道9

一切眾生，從淫欲而生。汝發心守貞修行，當須努力。倘有此等情念起，當思地獄刀山劍樹鑊湯爐炭種種之苦，自然種種念起，立刻消滅。每見多少善女，始則發心守貞不嫁，繼則情念一起，力不能勝，遂與人作茍且之事。而一經破守，如水潰堤，從茲橫流，永不能歸於正道，實可痛惜。當自斟酌，能守得牢則好極。否則出嫁從夫，乃天地聖人與人所立之綱常，固非不可也。（三）復陳蓮英書

一切眾生，從淫欲而生。你發心守貞不嫁修行，應當要努力。倘若有這些情思慾念生起，應當思惟地獄刀山劍樹，鑊湯爐炭的種種劇苦，自然種種慾念生起，立刻消滅。每每見到有多少善女子，開始是發心守貞不嫁，接著凡情慾念一生起，力量不能抵擋，於是與人作出苟且之事。而一旦破了貞節，就如同洪水潰堤，從此四溢橫流，永遠不能歸於正道，實在痛心可惜。應當自己斟酌衡量，能守得牢，當然非常好。否則，出嫁從夫，是天地聖人為人所建立的綱常倫理，當然是可以的。

# 勖居塵學道10

先後天衰弱，當以善於保養為事。若欲靠食物滋養，食素人宜多吃麥。食麥之力大於米力不止數倍。光吃了麵食，則精神健壯，氣力充足，音聲高大。米則只可飽腹，無此效力。麥比參力尚高數倍。有錢人服參，乃是錢無處用，故作此消耗耳。非真能補人也。又大磨麻油，亦補人。小磨麻油，以炒焦枯了，力道退半。人但知香，實則是焦味耳。蓮子，桂圓，紅棗，芡實，薏米，皆可滋補。豈必須血肉，方能滋補乎。總之皆不如麥之力大。（三）復蔡契誠書

先天、後天身體衰弱，應當以好好調養為事務。如果想靠食物來滋養，吃素的人應該多吃麥食。吃麥食的效力大過吃米的效力不只幾倍。我吃了麵食，就精神健壯，氣力充足，音聲高大。吃米飯只可以填飽肚子，就沒有這個效力了。麥子的效力比人參的效力還高好幾倍。有錢的人吃人參，這是有錢沒有地方用，所以要如此消費罷了。不是人參真的能滋補人。又有大磨麻油，也補人。小磨麻油，因為炒焦枯了，效力減退一半。平常人只知道小磨麻油很香，實際上是焦味罷了。蓮子、桂圓、紅棗、芡實、薏米，都可以滋補。哪裡必須要眾生的血肉，才能滋補呢？總之，都不如麥食的效力大。
